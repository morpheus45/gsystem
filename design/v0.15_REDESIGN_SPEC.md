# v0.15 — Refonte design "Indicator Calm"

Document de spécification AVANT codage.
Tu lis, tu valides ou tu corriges, ensuite je code.

## 1. Philosophie en une phrase

**Outillage de précision, pas startup tech.** L'app doit donner la même
confiance qu'un panneau de contrôle calibré : lisible immédiatement,
silencieuse, jamais décorative. Une seule couleur d'accent (ambre) pour
les signaux ; le reste est encre anthracite sur papier crème.

---

## 2. Palette finale

| Rôle | Couleur | Hex | Usage |
|---|---|---|---|
| Papier | crème laboratoire | `#F5EFE2` | fond de tous les écrans |
| Encre | anthracite | `#1A2438` | texte, contours, icônes |
| Encre douce | anthracite 60 % | `#1A2438` α 0.55 | sous-titres, hairlines |
| Encre hairline | anthracite 25 % | `#1A2438` α 0.25 | grille papier, séparateurs |
| Ambre signal | sodium | `#D89A3A` | UN SEUL accent : action active, badge MAJ, pip de statut |
| Erreur | rouge sourd | `#B23A3A` | uniquement erreur réelle |
| Succès | vert sourd | `#5A7A4D` | uniquement validation effective |

**Règle d'or :** un écran montre au maximum **une** zone ambre à la fois.
Si deux choses sont actives, l'une devient encre simple.

Polices à embarquer : **Tektur** (display) + **Geist Mono** (codes).

## 4. Composants

### Bouton primaire (Home, validation)
```
┌─────────────────────────────────────┐
│                                     │
│   TEMPS                             │
│   Feuille de temps (interventions)  │
│                                     │
└─────────────────────────────────────┘
```
- Hauteur 84 dp
- Coins **4 dp** (au lieu de 16 dp) — instrument, pas bulle
- Fond crème, contour encre **2 dp**
- Texte aligné à gauche, padding 20 dp
- Pas d'icône (la typographie suffit)

### Bouton actif / pip ambre
- Petit cercle ambre 8 dp **en haut-droite** du bouton concerné
- Apparait uniquement quand : badge MAJ disponible, action requise, statut critique

### Carte saisie (TEMPS, FRAIS, etc.)
```
┌──── G-S · FR / 054 ─────────── 17:32 ─┐
│                                       │
│  TEMPS                                │
│  ─                                    │
│                                       │
│   [contenu]                           │
│                                       │
└───────────────────────────────────────┘
```
- Header avec code de session (style dossier technique)
- Heure courante en haut-droite (instrument, pas montre)
- Ligne hairline horizontale sous le titre

### Champs de saisie
- Underline simple 1 dp (pas de cadre)
- Label flottant en Mono 11 sp
- Pas d'arrondis

### Chips / FilterChip
- Forme rectangulaire (radius 2 dp)
- État sélectionné : fond ambre, texte encre
- État non sélectionné : contour encre 1 dp, fond crème

## 8. Risques que je vois

1. **Crème sur écran AMOLED** = un peu d'énergie batterie. Acceptable
   pour une app de terrain qu'on ouvre 5 minutes max.
2. **Tektur** est une police gratuite Google Fonts, embarquée — pas de
   réseau requis.
3. **Lisibilité plein soleil** : contraste encre/crème (`#1A2438` sur
   `#F5EFE2`) = ratio ~13:1, largement WCAG AAA.
4. **Lisibilité cave/sombre** : pas de dark mode pour cette version
   (à voir en v0.16 si besoin). L'écran du téléphone fait la luminosité.
