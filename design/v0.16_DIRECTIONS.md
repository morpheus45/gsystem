# v0.16 — 3 directions design modernes 2026

Briefing : G-Systems doit ressembler à un outil de **terrain premium**
en 2026. Pas un manuel scolaire. Pas un panneau de musée. Pas un Material
Design générique qu'on a vu mille fois.

Référence mentale : Linear, Arc, Cluely, Apple Sport, Stripe, Tesla
cockpit, Revolut, ce que regarde un tech qui vient de quitter son
ancien employeur parce que leur app était nulle.

Tu choisis UNE direction. Je code en grand.

---

## DIRECTION A — "OBSIDIAN OPS"

**Mood :** Tableau de bord opérationnel premium. Sombre, dense, tactile.
Le contraire du Material 3 fade. Pense au cockpit d'un véhicule Tesla
de nuit, ou à l'app de monitoring d'un opérateur qui surveille un
réseau d'alarmes.

**Palette :**
| Rôle | Hex | Description |
|---|---|---|
| Fond | `#0A0B10` | noir profond avec une trace de bleu |
| Carte | `#16181F` | gris très sombre, lifté |
| Carte (active) | `#1F2230` | un cran plus clair |
| Hairline | `#2A2D38` | contour subtil des cartes |
| Texte primaire | `#F5F5F0` | blanc chaud, jamais pur blanc |
| Texte secondaire | `#8B92A8` | gris froid |
| **Signal** | `#FF3D5A` | **rouge signal alarme** — le métier |
| Signal halo | `#FF3D5A 30%` | glow autour des indicateurs actifs |

**Typo :** Tektur Bold 32sp pour les titres de boutons, Geist Mono pour
les numéros et codes (couleur signal sur active), corps en système.

**Bouton type :**
```
┌──────────────────────────────────────┐
│                                      │
│  01     TEMPS                        │
│         feuille de temps             │
│                                      │
└──────────────────────────────────────┘
  Carte sombre liftée, glow rouge sur tap, coins 12dp,
  numéro en mono rouge signal, label en Tektur 28sp blanc
```

**Pourquoi pour G-Systems :**
- Rouge = alarme, c'est le métier visuel des techs alarme
- Dark = OLED batterie économisée + œil reposé en cave/extérieur soir
- Plus de "pip" : on a une vraie barre de statut en haut avec contexte

## DIRECTION B — "DEEP STATION"

**Mood :** Station spatiale, cockpit Tesla de nuit, app Strava
premium. Bleu profond, accents cyan electric, glass effects légers,
gradients subtils. Vibe calme mais haute technologie.

**Palette :**
| Rôle | Hex | Description |
|---|---|---|
| Fond | `#0A1628` | bleu nuit profond |
| Carte | `#13243F` | bleu plus clair, lifté |
| Carte (active) | `#1E3559` | bleu net |
| Hairline | `#2C4974` | bleu hairline |
| Texte primaire | `#E8EEF7` | blanc bleuté |
| Texte secondaire | `#7A93B8` | bleu désaturé |
| **Signal** | `#00D9FF` | **cyan electric** — calme mais énergique |
| Signal halo | `#00D9FF 25%` | glow cyan |

**Typo :** Tektur Medium 30sp titres, Geist Mono codes en cyan, corps
système.

**Bouton type :**
```
╔════════════════════════════════════════╗
║                                        ║
║  ◢ 01    TEMPS                  →      ║
║          Feuille de temps              ║
║                                        ║
╚════════════════════════════════════════╝
  Gradient subtil de gauche à droite (plus sombre à droite),
  petit triangle ◢ cyan à gauche du numéro,
  flèche → discrète à droite
```

**Pourquoi pour G-Systems :**
- Évoque les systèmes de monitoring sécurité haut de gamme
- Cyan = numérique propre, pas alarmiste (différencie du rouge des sirènes)
- Premium sans être agressif

**Mood :** Warm dark refiné — Granola app, Notion dark, Arc warm
mode. Charbon chaud + grain papier subtil + ambre comme accent
artisanal. Premium artisan, pas tech-bro.

**Palette :**

| Fond | `#1A1814` | charbon chaud, légèrement brun |

| Grain | `#FFFFFF 4%` | texture papier subtile sur fond |
| Hairline | `#3D352F` | hairline chaude |

| Texte secondaire | `#A89880` | beige |

| Signal halo | `#FFB347 25%` | halo doux |

**Typo :** Tektur Medium 30sp en ivoire, Geist Mono codes en ambre,
corps système.

**Bouton type :**
```

┃  01·   TEMPS                         ┃
┃        Feuille de temps              ┃
┃                                      ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

**Pourquoi pour G-Systems :**
