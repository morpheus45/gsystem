🚀 G-Systems, en 1 phrase

Avant : Excel, mails, Viber, photos partout — la galère. Maintenant : un écran, 10 boutons. Le technicien fait son terrain, l'application fait la paperasse (Excel rempli, photos jointes, mails et Viber prêts, PV signé). Il ne reste qu'à appuyer sur Envoyer. ✨

⚡Démarrer (une seule fois)

1
📲 Installer l'application

Appuyez sur le gros bouton ci-dessus : le navigateur télécharge le fichier.

Ouvrez le fichier téléchargé et appuyez sur Installer. L'application est signée Google Play — aucune manipulation de sécurité à faire.

2
✍️ Renseigner vos informations

Au 1ᵉʳ lancement, 3 champs sont obligatoires (sinon l'application reste bloquée sur Réglages) :

• Votre nom   • Votre code tech (ex ISTGSxxx)   • L'email de votre responsable de secteur

Ajoutez aussi votre plaque (elle sert à nommer les photos). Les adresses du bureau et d'EPS sont déjà intégrées — rien à saisir, aucune erreur possible.

3
📊 Préparer votre fichier Excel

⬇️ Télécharger la trame officielle (.xlsm)

⚠ La trame est à ouvrir UNE FOIS sur un ordinateur (vrai Excel, pas le téléphone) : ouvrez-la, signez-la et enregistrez. Cette première ouverture active les macros — sans elle, le remplissage automatique ne fonctionne pas.

Placez ensuite le fichier sur votre téléphone (OneDrive, Drive ou Téléchargements). Son nom n'a pas d'importance.

4
🔗 Lier le fichier à l'application

Accueil → ENVOI MENSUEL → « Choisir mon fichier .xlsm » → sélectionnez votre fichier. Le choix est mémorisé — l'opération ne se fait qu'une seule fois.

✅ C'est prêt. Une boîte mail (Gmail/Outlook) et Viber installés sur le téléphone, et tout est en place pour travailler.

⬇️ Télécharger Viber (gratuit)
Viber sert aux messages d'équipe (clôtures, courrier, attente client). Installez-le et connectez-vous avec votre numéro avant la première intervention.

📝Faire signer un PV caméras

Deux tuiles à ne pas confondre

  • 🎥 Demande caméra = envoie une demande de rappel à EPS (n° de site + nombre de caméras), pour qu'un rendez-vous d'installation caméras soit reprogrammé.
  • 📝 PV caméras = le procès-verbal d'installation à remplir et faire signer sur place.

À quoi sert le PV ? Il pallie l'absence des documents au départ : le client veut des caméras alors que ce n'était pas prévu, donc le PV papier n'est pas dans le véhicule. Il se remplit et se signe (client + technicien) directement depuis le téléphone.

Aucun document n'est conservé dans l'application. C'est l'envoi du PDF par mail au client qui sert de preuve : le client le reçoit directement. Ensuite, depuis les « mails envoyés », imprimez le PDF pour l'adresser immédiatement à EPS, puis effacez le mail — rien n'est gardé.

1

Accueil → PV CAMÉRAS. Remplissez les informations client : convention, site, nom, adresse, date de souscription.

2

Cochez la prestation (installation initiale, mise en service anticipée, repose, caméra supplémentaire), saisissez les montants (caméras, cartes SD, abonnement, frais) et vos observations.

3

Faites signer à l'écran : le client dans « Signature de l'Abonné », le technicien dans « Signature du technicien ». Bouton Effacer pour recommencer.

4

Renseignez l'e-mail du client → « Générer et envoyer ». Le PDF rempli et signé par les deux parties s'ouvre dans l'app mail, destinataire déjà rempli → Envoyer.

5

Depuis les mails envoyés, imprimez le PDF pour l'envoyer immédiatement à EPS, puis effacez le mail — rien n'est conservé.

Le PV reproduit le document officiel à l'identique : seuls les champs sont à remplir, les signatures et les croix se posent automatiquement au bon endroit.

💡2 trucs à comprendre

🎁 GESTE CO : « Installé » ≠ « Offert »

  • Installé = ce qui est posé en plus du kit de base (les extensions uniquement). C'est ce qui compte pour les primes.
  • GESTE CO = ce qui est offert au client (part dans le mail client).
  • 📷 Caméras : seules les caméras posées lors d'une INSTALLATION comptent dans la prime. Une caméra posée sur un autre type d'intervention n'est pas primée.

⚠ Renseignez TOUJOURS « Installé », même sans rien offrir — sinon la prime du mois sera fausse. Pas de geste offert = aucun mail client, mais l'entrée et les primes sont bien enregistrées.

⏰ Les heures se calculent toutes seules

Plus aucune saisie d'heures : l'application les déduit des interventions du jour :

  • Matin et après-midi, les deux en OK → 8h
  • Matin et après-midi mais au moins une NR (1 OK + 1 NR, ou 2 NR) → 6h
  • Une seule demi-journée (OK ou NR) → 4h
  • Aucune intervention → 0h

L'écran TEMPS affiche le total du jour en direct. À l'ENVOI MENSUEL, les heures partent toutes seules dans l'Excel.

🧾 Tickets de frais — bon à savoir

  • MOBILE (forfait téléphone) : remboursé à 50 %, plafonné à 20 €. L'appli calcule le montant remboursé.
  • PARKING : un interrupteur TVA OUI / NON apparaît — mets NON pour les plateformes sans TVA (ex : PayByPhone).
  • Touchez la vignette d'un ticket (ou l'icône 👁) pour rouvrir la photo ou le PDF.
  • À l'ENVOI MENSUEL, les frais (TTC + TVA) se posent aussi dans l'Excel.

🔄Mises à jour

Quand une nouvelle version sort, revenez sur cette page et retéléchargez via le bouton : l'installation se fait par-dessus l'ancienne, les données sont conservées.

🛠️Si ça coince

Le sélecteur ne trouve pas le .xlsm → vérifiez qu'il est bien sur le téléphone (pas seulement sur le PC). Sur OneDrive, ouvrez-le une fois dans l'app OneDrive pour le mettre en cache.

Le mail s'ouvre sans destinataire → une app mail installée et connectée (Gmail, Outlook) est nécessaire.

Viber ne s'ouvre pas → vérifiez que Viber est installé (télécharger Viber), puis choisissez-le dans la liste des apps.

« Android 8.0 requis » → le téléphone est trop ancien (modèle antérieur à ~2018).
